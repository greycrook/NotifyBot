# Telegram-бот дежурств

## Запуск (Windows)

1. `setup.bat` — один раз
2. Заполните `BOT_TOKEN` в `.env`
3. `start.bat`

## Команды

**Для всех:** `/list`, `/today`, `/schedule`, `/help`

**Для админов группы:** `/setchat`, `/add`, `/remove`

### Добавление дежурного

```
/add @user1 @user2 @user3
/add 123456789
```

Несколько юзернеймов через пробел добавляются в очередь именно в указанном порядке.

ID можно узнать у [@userinfobot](https://t.me/userinfobot).

## Напоминания

**Боевой режим** (`TEST_MODE=false`): 10:00 и 17:00 по Москве, только в будни РФ.

**Тестовый режим** (`TEST_MODE=true`): время задаётся в `.env`:

```env
TEST_MORNING_HOUR=15
TEST_MORNING_MINUTE=20
TEST_EVENING_HOUR=15
TEST_EVENING_MINUTE=22
```

В тестовом режиме напоминания идут и в выходные. После смены `.env` перезапустите бота.

В логах видно, отправлено напоминание или почему пропущено.
